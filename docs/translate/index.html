Translate Ultroid to your native language

You can help in translating Ultroid into your native language!

Crowdin​

  • You can help translate Ultroid in Crowdin Platform.

  • URL : crowdin@Ultroid

  • Note: If your language isn't listed on that list, you can follow Github way..

Directly on Github​

  • You can download en.yml file from Github Repository.
  • You can translate it to any language you want.
  • File should be named in format : langcode.yml
  • Now, You can make a Pull Request to the Official Github Repo.
